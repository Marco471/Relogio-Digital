


# ⏰ Relógio Digital (Digital Clock)

Aplicação web desenvolvida com **HTML, CSS e JavaScript**, com suporte a **PWA (Progressive Web App)** 📱.

---

## 🚀 Funcionalidades

* ⏱️ Relógio digital em tempo real
* 📅 Exibição de data e dia da semana
* 📱 Design responsivo (funciona no celular)
* 📲 Instalável como aplicativo (PWA)
* ⚡ Funciona offline

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* PWA (Manifest + Service Worker)

---

## 📦 Estrutura do projeto

```
Relogio-Digital/
│
├── index.html
├── style.css
├── script.js
├── manifest.json
├── icon.png
├── icon-192.png
├── icon-512.png
└── unregisterServiceWorker.js
```

---

## 📲 Como instalar no celular

1. Abra o site no navegador (Google Chrome)
2. Toque em **"Adicionar à tela inicial"**
3. O app será instalado como um aplicativo 📱
