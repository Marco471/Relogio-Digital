# ⏰ Relógio Digital (Digital Clock)

Aplicação web desenvolvida com HTML, CSS e JavaScript, com suporte a PWA 📱.

## 🚀 Funcionalidades

- Relógio em tempo real  
- Data e dia da semana  
- Responsivo  
- Instalável no celular  
- Funciona offline  

## 🛠️ Tecnologias

- HTML  
- CSS  
- JavaScript  
- PWA  

## 📲 Como instalar

1. Abra no Chrome  
2. Clique em "Adicionar à tela inicial"
